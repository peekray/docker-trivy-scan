# Docker Trivy Vulnerability Scanner

## Descripción

Proyecto DevSecOps donde se construye una imagen Docker de una aplicación web simple y se realiza un escaneo de vulnerabilidades utilizando Trivy.

El objetivo es detectar vulnerabilidades conocidas (CVEs) dentro de la imagen del contenedor antes de desplegarla en un ambiente de producción.

---

## Tecnologías utilizadas

- Docker
- Trivy
- NGINX
- Ubuntu Linux
- GitHub
